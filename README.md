# BlackFOXtool
BlackFox is basic hacking tool for everyone !

# install packages < linux > and run
```
cd BlackFOXtool
bash install.sh
python3 blackfox.py
```
# updates
```
bash updates.sh
```
# install packages < windows > and run
```
pip3 install ipwhois
pip3 install bs4
pip3 install requests
cd BlackFOXtool
python3 blackfox.py
```
