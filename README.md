# BlackFOXtool
BlackFox is basic hacking tool for everyone !

# install < linux && MacOs >
```
git clone https://github.com/blackfox-Ctrl4/BlackFOXtool.git
```
# install packages < linux && MacOs > and run
```
cd BlackFOXtool
bash install.sh
python3 blackfox.py
```
### unzip sqlmap
Debian , Kali, Ubuntu ...
```
sudo apt install unzip
unzip sqlmap.zip
```
Fedora , CentOs ...
```
sudo yum install unzip
unzip sqlmap.zip
```
# updates
```
bash updates.sh
```
